# 🤖 Emma AI Interviewer

Emma AI Interviewer is an AI-powered mock interview platform that provides a realistic interview experience using voice interaction. Candidates can upload their resume, choose a target role, and participate in personalized HR and technical interviews generated by an AI interviewer.

---

## 🚀 Features

- 📄 Resume Upload (PDF/TXT)
- 🤖 AI-Powered Interviewer (Emma)
- 🎯 Resume-Based Question Generation
- 🎤 Voice-Based Interview
- 🔊 Text-to-Speech AI Responses
- 🎙️ Speech-to-Text Candidate Answers
- 💼 HR & Technical Interview Rounds
- 🔄 Real-Time Conversation Flow
- ☁️ Cloud Deployment (Render + Vercel)

## ⚙️ Workflow

1. Candidate uploads a resume and selects a target role.
2. The frontend sends the resume to the FastAPI backend.
3. The backend extracts resume text using PyPDF.
4. A prompt is generated based on the resume and job role.
5. The prompt is sent to the Groq Llama 3.1 model.
6. Emma generates personalized interview questions.
7. Questions are spoken using Text-to-Speech.
8. Candidate answers through Speech Recognition.
9. The backend maintains conversation history and generates follow-up questions.
10. The interview continues naturally and ends with a completion screen.
